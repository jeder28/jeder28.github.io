There's a board

  • Date: Oct 2015
  • Roles: Front-End, Back-End

There's a board is an internal app designed to encourage and aid Dev Bootcamp students to tutor by scheduling their time, providing goals to tutor a number of times and promote friendly competition between students on the number of times they tutored.

  • Implemented the jQuery plugin fullCalendar to neatly display scheduling information to our users.
  • Built our own front-end using Javascript to allow users to interact with our site.
  • Built the back-end using Rails and the Rails API to create and update appointments.
  • Used Bootstrap theme for a clean and unified style througout the site.
  • Deployed using Heroku

Stack Overphil

  • Date: Oct 2015
  • Roles: Built Custom CSS for Front End, Back-End

Stack Overphil is a mini-version of Stack Overflow that allowed our users to ask questions, answer questions, decide which answer to their own question was the best and vote on any question, answer or comment. We were then able to then sort this information by popularity, recently posted or by tags.

  • Back-end was built using Rails, to create questions, answers and comments and their corresponding relationships.
  • We implemented AJAX for quick filtering and searching for questions.
  • I used plain CSS for the styling of the site
